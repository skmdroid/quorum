A panel of agents reviews your code.

Quorum routes a diff to specialist reviewers — security, performance, correctness, tests, style — in parallel, then a synthesizer consolidates everything into one verdict. Try it live below; it's running the pure-Python engine in WebAssembly.

This page runs the deterministic offline reviewer (a heuristic, no LLM) so the demo is free and key-safe. For real reviews by an LLM panel, run it locally: pip install -e . && quorum review --diff pr.diff --provider claude-cli — see the README for providers, .quorum.json config, and the eval benchmark.
